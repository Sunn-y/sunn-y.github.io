## React란


- UI를 위한 오픈소스 자바 라이브러리
- 프레임워크가 아님
- UI에 focus

### 왜 react인가?

- 페이스북이 리액트로 만들어짐
- 깃허브에서 100k
- huge 커뮤니티
- Declarative
  - 리액트에 원한느 것을 말해라. 그러면 actual UI가 빌드됨
  - 렌더링/업데이트 시 효과적으로 핸들링할 수 있음
- React-native => 모바일 애플리케이션

### Prerequisites

- HTML, CSS, JS
- JavaScript
  -'this' keyword, filter, map and reduce
- ES6
  - let / const
  - arrow functions
  - template leterals
  - default parameters
  - object literals
  - rest and spread operator
  - destructuring assignment
