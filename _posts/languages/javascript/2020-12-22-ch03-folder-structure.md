## 프로젝트 폴더구조

- package.json
  - dependencies
  - scripts
- node_modules
- public
  - index.html : 딱 하나있는 html. SPA(Single Page Application)
- src
  - index.js : index.htmld의 root에 App을 렌더링
  - App.js : 렌더링 될 앱
  - App.test.js : 유닛테스트
  - App.css : App.js의 css 파일
  - index.css
  - serviceWorker.js : for progressive app
