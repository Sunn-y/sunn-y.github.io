## 준비사항 : node, text editor

    $ npx create-react-app hello-world
    $ cd hello-world
    $ npm start

### App.js의 내용 Hello World로 변경 -> 저장 -> 끝

## install 전역? or not?

### - npx

    $ npx create-react-app <project name>
    $ npm start

### - npm

    $ npm install create-react-app -g
    $ create-react-app <project name>
