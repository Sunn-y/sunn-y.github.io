##  \_Draft 폴더 만들기

- 프로젝트 폴더 바로 아래에 \_draft 폴더 생성
- 마크다운으로 포스트 작성

```
$ jekyll run serve
```

_포스트를 볼 수 없음_

```
$ jekyll run serve --draft
```
